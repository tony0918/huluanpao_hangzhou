2017 Run For Beauty 报名须知

一、比赛日期及项目

1. 比赛日期

2017年6月17日（星期六）

2. 比赛项目

（一）Lover - 12公里双人组，两人组队（组队形式：男女 / 男男 / 女女）；累计爬升476米；参赛名额150组（ 300人 ）

（二）Available - 22公里单人组 ，男女单人；累计爬升xxx米；参赛名额600人。

二、报名资格

（一）参加Lover - 12公里组和Avaiable - 22公里组比赛者须在比赛当年6月17日前满18周岁，身体健康，适宜参加该项比赛。

（二）Lover - 12公里组限报150队，Available - 22公里组限报600人，额满为止。

（四）按《 报名须知 》办理报名手续。

（五）以下疾病患者不宜参加 2017 Run For Beauty：先天性心脏病和风湿性心脏病患者 / 高血压和脑血管疾病患者 / 心肌炎和其它心脏病患者 / 冠状动脉病患者和严重心率不齐者 / 糖尿病患者 / 其他不适合运动的疾病患者。

三、报名时间

（一）2017年4月6日 - 2017年6月1日（ 报名参赛人数额满为止 ）；

（二）报名办法详见《 2017 Run For Beauty 报名须知 》。

四、报名方法

1. 登陆2017 Run For Beauty 赛事官方网站 http://hangzhou.huluanpao.com 进行报名

2. 关注“ 湖乱跑 ”官方微信，进入公众帐号，点击下方菜单“ 马上报名 ”进行报名。微信号：湖乱跑。

五、报名费

项目 报名费(所有国籍)
Lover - 12公里双人组 人民币 380元 / 队
Available - 22公里单人组 人民币 280元 / 人

六、计时芯片

所有组别赛事采用一次性芯片，赛后参赛者无须退还芯片。

七、在线报名流程及注意事项

1. 参赛者登陆2017 Run For Beauty 赛事官方网站 http://hangzhou.huluanpao.com 进行报名。或关注 “ 湖乱跑 ” 官方微信，进入公众帐号，点击下方菜单 “ 马上报名 ” 进行报名。微信号：湖乱跑。

2. 认真阅读报名须知、保险说明书及网上报名的相关内容。

3. 填写报名表。

4. 网上缴纳报名费（ 支持支付宝和微信支付以及信用卡）。

5. 付费成功后，即完成相应报名流程，报名注册邮箱将会收到确认邮件。完成报名后，可点击进入报名页面，点击“ 报名查询 ”，输入报名填写证件号查询报名状态。

6. 待报名截止后，选手可根据赛事官方公告相关说明登录赛事官方网站查询参赛号码和下载报名确认单。

7. 报名成功的参赛者于6月16日（ 周五 ） 10 : 00 – 20 : 00以及6月17日8：00 - 14：00，携带领物单和身份证原件至风雪户外狮虎桥店：杭州市下城区狮虎桥路8号领取比赛装备装备包括：号码布、计时芯片、背袋等。参赛包不允许代领。

八、报名确认后如不参加比赛，将不予退还报名费，参赛项目和其他一切个人信息经确认后不得更改，参赛名额不能转让。

九、本报名须知最终解释权归2017 Run For Beauty 赛事组委会所有。未尽事宜，另行通知。

2017 Run For Beauty 赛事组委会
